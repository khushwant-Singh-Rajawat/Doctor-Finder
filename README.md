# 🏥 Doctor Finder Web Application

## 📌 Project Overview
Doctor Finder is a full stack web application designed to help users easily find and connect with doctors based on their specialization and location. The system provides a simple and interactive interface for searching doctors and viewing their details.

---

## 🚀 Features
- 🔍 Search doctors by specialization
- 📍 Filter doctors by location
- ⚡ Fast data loading using AJAX
- 👨‍⚕️ View doctor details
- 💻 User-friendly interface

---

## 🛠️ Technologies Used
### Frontend:
- HTML
- CSS
- JavaScript
- AJAX

### Backend:
- Java Servlets
- JSP (Java Server Pages)

---

## ⚙️ How It Works
1. User enters search criteria (specialization/location)
2. AJAX sends request to server
3. Servlet processes request
4. Data fetched and returned to JSP
5. Results displayed dynamically on UI

---

## 📂 Project Structure
- `src/` → Java source code (Servlets)
- `webapp/` → JSP, HTML, CSS, JS files
- `WEB-INF/` → Configuration files (web.xml)

---

## ▶️ How to Run
1. Clone the repository:
2. Open project in IntelliJ IDEA
3. Configure Apache Tomcat server
4. Deploy and run project
